Для запуска проекта на локальном сервере потребуется XAMPP, PHP 8.1>=. 
Для запуска на XAMPP нужно скопировать репозиторий 'intersvyaz' в папку xampp\htdocs, зайти в эту папку и командую строку прописать команду compser install, запустить Apache сервер и перейти по ссылке http://localhost/intersvyaz/web/index.php?r=traffics
